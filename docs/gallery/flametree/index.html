Flametree

Fractious petals, stop interrupting me with your boring beauty
(Jennifer Chang)



I have a soft spot for the flametree package and series. The algorithm it uses is a simple L-system, and was easier to implement than I had been expecting. It was rather popular on twitter. Not only did other people use it, Tyler Morgan-Wall made a very cool 3d version called raybonsai. I’ve even managed to use it in my own teaching. The most bizarre thing, however, is the fact that I wrote the code in April 2020 and although I’m writing this summary a few months later, it feels like a lifetime ago.
